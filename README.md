## 📊 Projects

### 🔹 Task 1: Iris Dataset Analysis (EDA)
- Performed Exploratory Data Analysis (EDA) on the Iris dataset.
- Analyzed dataset structure, features, and statistical summary.
- Created visualizations including:
  - Scatter plots
  - Histograms
  - Box plots
  - Pair plots
- Identified relationships between features and different flower species.

---

### 🔹 Task 2: Loan Default Prediction
- Built a classification model to predict loan default.
- Handled missing values and encoded categorical variables.
- Used Logistic Regression for prediction.
- Evaluated model using:
  - Accuracy score
  - Confusion matrix
  - Classification report

---

### 🔹 Task 3: Customer Churn Prediction
- Developed a machine learning model to predict customer churn.
- Applied Random Forest classifier.
- Performed data preprocessing and feature encoding.
- Analyzed feature importance to identify key factors influencing churn.

---

### 🔹 Task 4: Insurance Cost Prediction
- Implemented a regression model to predict insurance charges.
- Used Linear Regression algorithm.
- Evaluated model performance using:
  - Mean Absolute Error (MAE)
  - Root Mean Squared Error (RMSE)
- Visualized relationships between features and insurance cost.

---

### 🔹 Task 5: Loan Acceptance Prediction
- Built a classification model to predict loan acceptance.
- Processed customer data and encoded categorical features.
- Used Logistic Regression for classification.
- Visualized loan acceptance distribution.
